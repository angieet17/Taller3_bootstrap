Guía básica de Git paso a paso

Paso 1: Verificar la instalación de Git

Comprueba que Git está instalado correctamente en tu equipo.

git --version

Paso 2: Configurar tu identidad

Define tu nombre y correo para identificar tus cambios.

git config --global user.name "Tu Nombre"
git config --global user.email "tu.email@example.com"

Paso 3: Inicializar proyecto local

Crea o entra a tu carpeta y actívala como repositorio.

cd ruta/a/tu/proyecto
git init

Paso 4: Agregar archivos y commit

Guarda tu primera versión del proyecto.

git add .
git commit -m "Primer commit"

Paso 5: Crear repositorio en GitHub

Crea el repositorio remoto.

  • Inicia sesión en GitHub
  • Crea un nuevo repositorio
  • Asigna nombre y visibilidad
  • Haz clic en “Create repository”

Paso 6: Conectar con GitHub

git branch -M main
git remote add origin https://github.com/tu_usuario/tu_repositorio.git
git push -u origin main

Paso 7: Publicar proyecto

Sube todo a GitHub.

git push
