# 🎨 Galería de Arte

Proyecto desarrollado con HTML, CSS y Bootstrap 5.

## 📌 Descripción

Sitio web inspirado en una galería de arte que incluye diferentes páginas y componentes de Bootstrap para practicar maquetación y diseño responsivo.

## 📂 Contenido

- 🏠 Página principal con carrusel y tarjetas.
- 📊 Tabla responsive con información sobre movimientos artísticos.
- 👤 Formularios de registro e inicio de sesión.
- 🐙 Página con guía paso a paso para usar Git y GitHub.
- 🔗 Botón para acceder al repositorio de GitHub.

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5.3
- Bootstrap Icons
- Git
- GitHub

## 📁 Estructura del proyecto

```
├── index.html
├── app
│   ├── tables.html
│   ├── forms.html
│   ├── github.html
│   └── repository.html
├── assets
│   ├── css
│   │   └── style.css
│   └── img
```

## 🚀 Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/angieet17/Taller3_bootstrap.git
```

2. Abre la carpeta del proyecto.

3. Ejecuta `index.html` en tu navegador.
